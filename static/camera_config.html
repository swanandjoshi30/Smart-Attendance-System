💡 Camera URL Guide

  • Blank (local webcam): Uses the browser's default camera (or second camera for out-cam).
  • IP Camera (MJPEG): Most IP cameras expose an HTTP MJPEG stream — use http://192.168.x.x/video or similar.
  • Mobile phone: Install an IP Webcam app (e.g. "IP Webcam" on Android). Use the MJPEG URL it provides.
  • CCTV / NVR: Many support RTSP. Use an RTSP-to-MJPEG proxy (e.g. go2rtc) and enter the resulting HTTP URL here.
  • Testing: Run one camera on your laptop (blank) and another on a mobile (HTTP URL). Both will show in the session view.
